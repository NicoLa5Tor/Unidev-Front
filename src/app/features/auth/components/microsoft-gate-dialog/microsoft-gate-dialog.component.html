Acceso institucional requerido

Para registrar tu correo dentro del tenant Microsoft de UniDev necesitamos validar que el dominio de tu institución (empresa, universidad o colegio) ya forme parte de nuestro ecosistema. Si tu dominio aún no está habilitado, lo integraremos después de recibir tu solicitud.

Correo institucional Usa únicamente dominios oficiales registrados por tu organización. Este campo es obligatorio. Ingresa un correo válido (usuario@dominio).
