② Ajuste de horas por nivel

horas = horasBase × 100 / eficiencia%
El 100 normaliza el porcentaje a decimal. Eficiencia 200% → ÷ 2 → mitad de horas. Eficiencia 100% → ÷ 1 → sin cambio.

③ Tarifa por hora

tarifa = SMLV ÷ horas/mes × (eficiencia / 100)
El SMLV y las horas mensuales los configura el admin. A mayor eficiencia, mayor tarifa — el perfil es más valioso por hora.

④ Monto base

base = horas × tarifa/h
Costo puntual sin margen. Un Senior cobra más por hora pero termina antes, por eso los montos base suelen ser similares entre niveles.

⑤ Rango comercial

mín = base × (1 − %/100) · máx = base × (1 + %/100)
El porcentaje de apertura cubre variabilidad real: cambios de alcance, bugs imprevistos y reuniones no planificadas.
