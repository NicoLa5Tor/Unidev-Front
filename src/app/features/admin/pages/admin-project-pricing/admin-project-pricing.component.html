Editar nivel existente

Ayuda

Los niveles son el catalogo base del pricing. Por defecto arrancan en `JUNIOR`, `MIDDLE` y `SENIOR`.

Las tarifas ya no inventan su propio tipo: siempre cuelgan de uno de estos niveles ya existentes.

La productividad 100% representa el baseline. Un valor mayor reduce las horas calculadas para ese nivel.
